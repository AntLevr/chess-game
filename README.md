# 领土战争.io

一个基于Web的领土战争策略游戏，类似Territorial.io玩法。

## 功能特点

- 50x50的大地图，支持6方对战（1个玩家 + 5个AI）
- 实时兵力自动增长机制
- 可调节的扩张速度（1%-100%）
- 智能AI对手
- 实时排行榜显示
- Canvas渲染，流畅的游戏体验
- 响应式设计，支持移动端

## 游戏规则

### 基本规则
- 游戏在50x50的地图上进行
- 玩家与5个AI对手争夺领土
- 最后存活的玩家获胜

### 兵力增长
- 每个领土每帧自动增长兵力
- 领土越多，增长速度越快
- 初始每个领土100兵力

### 扩张机制
- 点击己方领土选中
- 选中后会自动向相邻领土扩张
- 扩张速度可调节（1%-100%）
- 攻击敌方领土会消耗兵力
- 占领空地只需派出兵力

### 策略建议
- 不要把所有兵力都派出去，保留防守力量
- 前期避免与其他玩家正面冲突
- 优先扩张空地，建立优势
- 观察AI动向，抓住机会反击

## 如何运行

1. 克隆或下载项目到本地
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

- HTML5 Canvas

## 游戏操作

- **选择领土**：点击己方领土选中
- **自动扩张**：选中后会自动向相邻领土扩张
- **调节速度**：使用滑块调节扩张速度（1%-100%）
- **重新开始**：点击"重新开始"按钮重置游戏
- **查看规则**：点击"游戏规则"按钮查看详细规则
