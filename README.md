# 战旗游戏

一个基于Web的经典战旗游戏，支持双人对战。

## 功能特点

- 完整的战棋游戏规则实现
- 双人对战模式
- 棋子升级为王的功能
- 连续跳吃机制
- 响应式设计，支持移动端
- 美观的UI界面和动画效果

## 游戏规则

### 基本规则
- 游戏在8x8的棋盘上进行
- 红方先行，双方轮流移动
- 每方有12个棋子

### 移动规则
- 普通棋子只能向前斜向移动一格
- 到达对方底线后升级为"王"
- 王可以向前后任意斜向移动

### 吃子规则
- 当对方棋子在斜前方且其后方为空时，可以跳过吃子
- 被吃掉的棋子从棋盘移除
- 王可以向后跳吃
- 可以连续跳吃多个棋子

### 胜利条件
- 吃掉对方所有棋子
- 对方无法移动

## 如何运行

1. 克隆或下载项目到本地
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

## 游戏操作

- 点击己方棋子选中
- 绿色高亮显示可移动位置
- 红色高亮显示可吃子位置
- 点击高亮位置执行移动或吃子
- 点击"重新开始"按钮重置游戏
- 点击"游戏规则"按钮查看详细规则
