# 领土战争

一个基于Web的领土战争策略游戏，支持4方对战。

## 功能特点

- 10x10的地图，支持4方对战
- 增兵和攻击两个游戏阶段
- 骰子战斗机制
- 实时统计各玩家领土和兵力
- 响应式设计，支持移动端
- 美观的UI界面和动画效果

## 游戏规则

### 基本规则
- 游戏在10x10的地图上进行
- 支持4方对战：红方、蓝方、绿方、黄方
- 每方从初始领土开始扩张
- 最后存活的玩家获胜

### 增兵阶段
- 每回合开始时获得增兵
- 增兵数量 = 领土数 ÷ 3（最少3个）
- 点击己方领土放置增兵
- 必须用完所有增兵才能进入攻击阶段

### 攻击阶段
- 选择己方领土（兵力>1）
- 点击相邻敌方领土进行攻击
- 攻击方最多掷3个骰子
- 防守方最多掷2个骰子
- 点数大的一方获胜，败方损失兵力
- 防守方兵力归零则被占领

### 胜利条件
- 消灭所有其他玩家
- 成为最后存活的玩家

## 如何运行

1. 克隆或下载项目到本地
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

## 游戏操作

- **增兵阶段**: 点击己方领土放置增兵
- **攻击阶段**: 
  - 点击己方领土（兵力>1）选中
  - 绿色高亮显示可移动位置
  - 红色高亮显示可攻击位置
  - 点击高亮位置执行攻击
- 点击"结束回合"按钮结束当前回合
- 点击"重新开始"按钮重置游戏
- 点击"游戏规则"按钮查看详细规则
